# NetPractice - 42

*This project has been created as part of the 42 curriculum by **alejandj**.*

## 📝 Description

**NetPractice** is a project focused on learning the basics of computer networking.

The objective of this project is to understand IP addressing, subnetting, and routing by solving different network
configuration exercises. Each level presents a simulation of a network that must be correctly configured to ensure
communication between all devices.

## 🚀 Instructions

To use **NetPractice**, you first need to download the `.tgz` file provided by 42.

1. Extract the archive and run `./run.sh` to start the NetPractice server.
2. Open your browser and go to `http://localhost:49152` to access the interface.

In this interface, you can choose between two modes:

- **Training** → You can complete all levels from 1 to 10.
- **Evaluation** → You will be given 3 random exercises between levels 6 and 10 to prepare for the real evaluation.

After successfully configuring and solving each level, you must export your configuration by clicking
the "Get my config" button in the interface. This will download the specific configuration file for that level.

### 📦 Submission

- You must export a total of 10 configuration files (one per level).
- All files must be placed at the root of the repository.
